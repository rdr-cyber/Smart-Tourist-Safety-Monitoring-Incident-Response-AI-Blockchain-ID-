Core Features

  • Home Dashboard - Map with geo-fencing, status bar, and SOS button
  • Digital Identity - Verifiable Credential storage and QR code generation
  • Incident Reporting - AI detection and manual reporting options
  • Notifications - Real-time alerts and updates
  • Settings - Privacy preferences and offline mode

Newly Added Features

  • Itinerary Risk Scanner - Analyzes planned routes and flags risky segments
  • Integrated Digital Safe - Secure document storage with emergency sharing
  • Anonymous Reporting - Report incidents without revealing identity
  • AI-Based Safety Ratings - Location safety scores based on data analytics
  • Guide/Taxi Detector - Verification system for tour guides and taxis
  • Travel Insurance Integration - Quick claims and policy management

Technical Implementation

The app is built with React Native for cross-platform compatibility. Key technologies include:

  • React Navigation for seamless screen transitions
  • React Native Maps for interactive geofencing
  • Styled Components for consistent UI design
  • Local storage for offline functionality
  • Mock data for demonstration purposes
